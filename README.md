# Wedding Invitation Website

A cinematic, luxury-style wedding invitation website built using pure HTML, CSS, and JavaScript.
This project focuses on elegant typography, immersive animations, floating flower petals, smooth scrolling, ambient audio controls, and premium visual aesthetics.

## 🌸 Features

- Premium Visual Experience
- Floating Flower Petal Animation
- Ambient Music System

---

### Wedding Details
- Date
- Time
- Venue information

### Love Story Timeline
- Animated relationship journey section

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
- SVG Artwork
- Google Fonts
- Google Maps Embed API
